# Yogesh Lovevanshi - Portfolio Website

A modern, fully responsive developer portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- 🎨 Modern and clean UI design
- 🌓 Dark/Light mode toggle
- 📱 Fully responsive (mobile, tablet, desktop)
- ✨ Smooth animations with Framer Motion
- 🎯 SEO friendly
- 💼 Professional sections: Hero, About, Skills, Projects, Experience, Resume, Contact

## Tech Stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Customization

1. Update personal information in components
2. Replace project images and links in `Projects.jsx`
3. Add your resume PDF to `public/resume.pdf`
4. Update social media links throughout components
5. Modify color scheme in `tailwind.config.js`

## Sections

- **Hero**: Introduction with name, title, and CTA buttons
- **About**: Professional introduction and focus areas
- **Skills**: Categorized technical skills with icons
- **Projects**: Featured projects with images and links
- **Experience**: Learning journey timeline
- **Resume**: Download resume section
- **Contact**: Contact form and information
- **Footer**: Copyright and social links
